# AI vs Human Text Detector
This project detects whether a given text is written by a Human or generated by AI using Machine Learning and NLP techniques.
It uses text preprocessing, TF-IDF vectorization, and trained ML models (Logistic Regression and SVM) to classify text.

# Features
* Detects AI-generated vs Human-written text
* Uses NLP preprocessing: cleaning, stopword removal, contractions handling
* TF-IDF based text representation
* Trained using Logistic Regression and SVM
* Web interface built using Flask, HTML, and CSS

# Tech Stack
* Python
* Pandas, NumPy
* Scikit-learn
* NLTK, SpaCy
* Flask
* HTML, CSS

# Model Details
* Input: Raw text
* Preprocessing: Cleaning, stopwords removal, contraction expansion
* Vectorization: TF-IDF
* Models Used:
* Logistic Regression
* Support Vector Machine (Final model saved)
* Accuracy achieved: ~99% on test data.
